## 🛠️ Технологический стек

- **HTML5 и CSS3** — вёрстка адаптивных интерфейсов.
- **JavaScript** — базовый язык разработки.
- **TypeScript** — строго типизированный JavaScript.
- **React** — создание современных пользовательских интерфейсов.
- **Bootstrap 5** — фреймворк для быстрой разработки адаптивных веб‑интерфейсов.
- **CI/CD** — автоматизация процессов интеграции и доставки кода.
- **Sass/SCSS** — препроцессор CSS для структурирования и упрощения работы со стилями.
- **Playwright** — инструмент для автоматизации тестирования веб‑приложений и end‑to‑end‑тестов.
- **Postman** — работа с API: тестирование, документирование и отладка запросов.
- **Git** — система контроля версий для отслеживания изменений в коде и совместной работы.

## 📊 Проекты

В этом разделе представлены мои лучшие работы. Каждый проект имеет подробное описание в соответствующем репозитории:

- **🧠 «Игры разума»** — консольное приложение, представляющее из себя набор из пяти простых математических игр. Они построены по принципу популярных мобильных приложений для прокачки мозга.  
  **СТЕК:** Чистый JS с использованием стандартных конструкций языка: функции, циклы, условные операторы.  
  [GitHub](https://github.com/GromoZeus/frontend-project-44)

- **⚙️ «Вычислитель отличий»** — консольное приложение, представляющее из себя программу, которая определяет разницу между двумя структурами данных. Утилита поддерживает входные форматы YAML и JSON, парсит и обрабатывает данные из них и выводит результат.  
  **СТЕК:** Node.js  
  [GitHub](https://github.com/GromoZeus/frontend-project-46)

- **📡 RSS‑агрегатор** — веб‑приложение для сбора и просмотра информации RSS‑лент из различных источников. Оно позволяет добавлять неограниченное количество RSS‑лент, само их обновляет и добавляет новые записи в общий поток.  
  **СТЕК:** JS, Valtio, Axios, i18next, Bootstrap  
  [GitHub](https://github.com/GromoZeus/frontend-project-11)

- **💬 Чат‑приложение** — веб‑приложение для обмена мгновенными сообщениями в реальном времени. Проект включает регистрацию новых пользователей, добавление и удаление новых каналов (аналог очень упрощённой версии Slack‑чата).  
  **СТЕК:** React, Redux Toolkit, React Bootstrap & Bootstrap, WebSocket  
  [GitHub](https://github.com/GromoZeus/frontend-project-12)
